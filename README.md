# Household Dashboard

A full-stack household operations dashboard built to organize recurring tasks, surface important information, and provide a cleaner view of day-to-day home workflows.

## Overview

This project explores the same product thinking used in internal business tools: centralizing information, reducing friction, and making routine workflows easier to manage.

The app is built with a modern Next.js frontend and supporting backend functionality, with an emphasis on reusable UI components, clear layouts, and maintainable structure.

## Why I Built It

I wanted to build a dashboard-style application that feels closer to a practical internal tool than a tutorial project. The goal was to practice application structure, component reuse, and workflow-oriented UI design in a way that translates well to business software.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
